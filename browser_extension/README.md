# FocusOne Guard (브라우저 확장)

FocusOne 데스크탑 앱의 동반 확장 프로그램. 데스크탑 앱과 독립적으로 **상시 동작**하며, 팝업에서 기능별로 켜고 끌 수 있다.

## 기능

| 기능 | 설명 | 기본값 |
|---|---|---|
| 영상 잠금 | 유튜브 영상을 재생하면 **끝까지 보기 전에는 다른 영상으로 이동 불가**. 링크 클릭, "다음 영상" 버튼, 주소창 이동 전부 방어 | 켜짐 |
| 앞으로 감기 차단 | 이미 시청한 지점보다 **앞으로 시킹 불가** (화살표 키 포함). 뒤로 돌려 다시 보는 건 허용 | 켜짐 |
| 중복 탭/창 차단 | 등록한 사이트(기본: youtube.com)는 **한 번에 한 탭만**. 새 창을 띄워 다시 들어가는 우회도 차단 — 새 탭이 닫히고 기존 탭으로 이동됨 | 켜짐 |

- 영상 끝 3초 전부터는 "다 본 것"으로 간주하고 잠금이 풀린다
- 잠금이 풀리는 유일한 다른 방법은 팝업에서 기능을 끄는 것

## 설치 (Chrome / Edge)

1. `chrome://extensions` (Edge는 `edge://extensions`) 접속
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 이 `browser_extension` 폴더 선택

설치 후 툴바의 확장 아이콘을 클릭하면 설정 팝업이 열린다.

## 파일 구성

```
browser_extension/
├── manifest.json     # Manifest V3 정의
├── background.js     # 중복 탭/창 차단 (서비스 워커)
├── youtube-lock.js   # 영상 잠금 + 앞으로 감기 차단 (콘텐츠 스크립트)
├── popup.html        # 설정 팝업 UI
└── popup.js          # 설정 저장/로드
```

## 알아둘 것

- 데스크탑 앱의 "차단 탭"(집중 시간 한정 소프트 차단)과는 별개 기능이다. 이 확장은 시간과 무관하게 항상 동작한다
- 확장을 지우거나 브라우저 개발자 모드를 끄면 무력화된다 — 완전한 강제가 아니라 "이탈 비용을 높이는" 도구다
- 유튜브 UI 개편으로 셀렉터(`ytp-next-button` 등)가 바뀌면 업데이트가 필요할 수 있다
- Firefox는 Manifest V3 지원 차이로 현재 미지원 (Chrome/Edge/Whale 등 Chromium 계열 대상)
